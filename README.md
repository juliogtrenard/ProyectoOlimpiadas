# Título del proyecto

ProyectoOlimpiadas

## Descripción

Aplicación para gestionar datos de olimpiadas. Trata deportistas, participaciones, olimpiadas, deportes, eventos y equipos.

## Primeros pasos

### Cómo usar

* Clona el repositorio en algún IDE y ejecuta el programa.
